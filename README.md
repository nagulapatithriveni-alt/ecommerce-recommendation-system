# E-Commerce Recommendation System

## Overview

This project implements an E-Commerce Product Recommendation System using Machine Learning. The system analyzes customer purchase behavior and recommends similar products based on historical buying patterns.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn
* Joblib
* Jupyter Notebook

## Methodology

1. Data Cleaning and Preprocessing
2. Customer-Product Matrix Creation
3. Product Similarity Calculation
4. Cosine Similarity Analysis
5. Product Recommendation Generation

## Features

* Customer purchase analysis
* Product similarity detection
* Recommendation generation
* Top-selling product analysis
* Data visualization

## Example Recommendation

Input Product:
4 PURPLE FLOCK DINNER CANDLES

Recommended Products:

* 4 PINK DINNER CANDLE SILVER FLOCK
* 4 BLUE DINNER CANDLES SILVER FLOCK

## Project Outcome

A recommendation engine capable of suggesting similar products using customer purchasing behavior and cosine similarity techniques.
