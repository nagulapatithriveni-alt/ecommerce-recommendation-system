# E-Commerce Recommendation System

## Overview

This project implements an E-Commerce Product Recommendation System using Machine Learning and Cosine Similarity. The system analyzes customer purchase history and recommends similar products based on customer buying behavior.

## Dataset

* Online Retail Dataset
* 397,924 cleaned transactions
* 3,877 unique products

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Joblib
* Jupyter Notebook

## Methodology

1. Data Cleaning
2. Customer-Product Matrix Creation
3. Product Similarity Calculation
4. Cosine Similarity Analysis
5. Product Recommendation Generation
